# task1
This project is a simple Login and Dashboard UI built using HTML, CSS, and JavaScript.  
The focus is on clean design, usability, responsiveness (desktop & mobile), and basic client-side validation.

Login Page
- Email and Password input fields  
- Client-side validation (required fields, email format)
- Error and success messages
- Redirects to dashboard on successful login

Dashboard Page
- Header with application name and profile icon
- Sidebar navigation (Dashboard, Profile, Settings, Logout)
- Main content area with welcome message
- Informational cards (Users, Orders, Revenue)
